## Название проекта: 
Витрина данных в Яндекс Такси.

## Описание проекта: 
Построение витрины данных, которая будет агрегировать информацию о поездках по каждому способу оплаты. 
Написание PySpark-скрипта для расчета ключевых показателей (количество поездок, средняя стоимость поездки, средние чаевые и суммарную выручку по каждому типу оплаты) 
и создание DAG в Airflow для автоматизации процесса создания таблицы, которая станет основой для финансовых отчётов и аналитических дашбордов.

Использованные библиотеки: pyspark.sql, datetime, airflow, airflow.sensors.s3_key_sensor, airflow.providers.yandex.operators.dataproc
